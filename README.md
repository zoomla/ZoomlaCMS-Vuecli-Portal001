# ZoomlaCMS-Vuecli-Portal001
基于ZoomlaCMS+VueCli门户网站项目，在v4传统门户基础上引伸改进的项目，方便传统web设计者学习掌握，下载即可用。

http://wwww.z01.com

#使用方法

##第一步安装依赖
```
npm install
```

##第二步 运行项目
```
npm run server
```

##获取支持
产品官网：http://www.z01.com

使用手册：http://help.z01.com
